## 🔧 手動設定コマンド（代替方法）

もしGUIでの設定がうまくいかない場合は、コマンドラインで設定できます：

```bash
# 手動でIPアドレスを設定
sudo networksetup -setmanual "Wi-Fi" 192.168.151.100 255.255.254.0 192.168.150.1

# DNS設定
sudo networksetup -setdnsservers "Wi-Fi" 8.8.8.8 8.8.4.4 1.1.1.1

# 設定確認
networksetup -getinfo "Wi-Fi"
```

## 📋 設定完了後の再テスト手順
